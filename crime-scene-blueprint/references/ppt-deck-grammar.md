# PPT Deck Grammar

Use this reference when creating a slide outline or host-led flow.

## Standard Slide Flow

| Phase | Typical Slides | Purpose |
| --- | ---: | --- |
| Cover | 1 | Episode identity and tone |
| Incident setup | 2-3 | What happened and what must be solved |
| Victim/object | 4 | Victim, missing person, stolen item, or target |
| Suspects | 5-8 | Four suspects with name and role |
| Alibi | 9 | Alibi prompt before evidence |
| Evidence collection intro | 10-11 | Searchable rooms/items checklist |
| Location search | 12-15 | Room maps with numbered hotspots |
| Suspect belongings | 16-19 | Personal evidence entry points |
| Evidence cards | 20-49/63 | Numbered clues, contradictions, red herrings |
| Optional gate | variable | Password, lab request, final search, or result check |
| Accusation | near end | Vote or culprit nomination |
| Culprit reveal | near end | Culprit name |
| Case truth | final slides | Chronological explanation and decisive proof |

## Information Gates

Protect pacing with gates:

1. Premise and alibis only.
2. Location search.
3. Suspect belongings.
4. Numbered evidence.
5. Optional synthesis puzzle, lab request, password, or final check.
6. Accusation.
7. Truth and decisive evidence.

Use an optional gate only when it forces synthesis. Do not add a password, lab, or final check as decoration.

## Evidence Reveal Order

Early:
- establish scene,
- introduce suspicious objects,
- seed multiple possible culprits.

Middle:
- contradict alibis,
- reveal hidden relationships,
- connect locations to suspects.

Late:
- tie method to movement,
- expose staging or concealment,
- clear innocent suspects.

Final:
- make the culprit unavoidable,
- replay the real sequence,
- explain why red herrings were misleading.

## Host Flow

For each slide or clue, track:

| Slide | Host Action | Player Sees | Author-Only Purpose | Unlocks |
| --- | --- | --- | --- | --- |
| | read premise / show evidence / ask vote | | suspicion / contradiction / proof / clearance | |

Never place author-only truth on a player-facing slide before the accusation.
